<처음 로컬과 원격을 연동할 시>

- github에 repo(원격)만들기
- 원격과 연동할 로컬 저장소 폴더 생성
- 로컬저장소 폴더경로를 복사하여 cmd에서 cd하기
- 해당 경로에 들어가, git clone + "원격저장소 주소" 
- 그러면 로컬에 원격저장소이름의 폴더가 생성될 것이다.
- 그럼 그 폴더로 들어가 새 문서를 편집/추가 한다.

< 새로운 문서 추가: 로컬에서 추가하여 원격으로 push>
- 해당 로컬의 폴더에 새로운 문서를 추가한다.
- git status하면 새로이 추가되었다고 뜰 것임.(초록글씨)
- git add .(모든 파일을 추가함) 또는 git add filename 을 한다.
- git commit -m " add ~~~~ [add]" 등의 문구를 넣어 설명 붙임.
- 마지막으로 git push를 하여 로컬->원격으로 반영시킨다. 그럼 끝!


<기존 문서 변경: 로컬에서 변경하여 원격으로 push할 때>

- 해당 로컬의 폴더에서 파일을 수정한다.
- git status하면 파일이 modify되었다고 뜰 것임.(빨간글씨)
- git add 로 선택/또는 전체 파일을 원격으로 민다.
- git commit -m " add ~~~~ [add]" 등의 문구를 넣어 설명 붙임.
- git push 마지막으로 한다.
